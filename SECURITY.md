# Security Policy

## Supported Versions

We actively support the latest released version of **survy**.

Older versions may not receive security updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately by contacting the maintainer.

Provide as much detail as possible, including:

* Description of the issue
* Steps to reproduce
* Potential impact

---

## Response Process

* We will acknowledge your report as soon as possible
* We will investigate and work on a fix
* A patched version will be released if necessary

---

## Disclosure

We ask that you do not publicly disclose the vulnerability until a fix has been released.

---

Thank you for helping keep **survy** secure 🙏
